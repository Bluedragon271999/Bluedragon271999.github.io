Python Calculator

About the project

This is project is a culmination of 6 different simple calculators made in the programming language python. The point of this project was to create a program using something not taught at PTC. This project is a console program and was made using Visual Studio 2019. It has the user select the calculator that he or she wants, then gets input from the user based on the used calculator. The program is continuous until the user inputs 999 to finish using the calculators.
