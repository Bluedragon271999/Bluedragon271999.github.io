About the project

This project is a website meant to hold stat blocks for npcs and enemies for the table top role-playing system Genesys. You can create your own stat blocks or you can use a stat block that is already created and on the website. This website is meant to help Game Masters with organizing his or her enemies. The website also allows you to search through the stat blocks by npc or enemies name.
