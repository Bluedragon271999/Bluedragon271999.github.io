Gem Miner

About the project

This project is the updated version of my game. The game was given a new title, "Gem Miner". Almost all of the graphics have been changed in the game. The game was given a cave aesthetic because it seemed to fit the game properly. The game was given a new mechanic, stage hazards. The game now has spikes that the player must avoid in order to survive the game. The instuctions of the game was updated so that it was accurate and informative to the player.
